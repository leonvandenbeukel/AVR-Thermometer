# AVR-Thermometer

This is an AVR version of a thermometer. The used components are:

1x ATMega328P (not an arduino)

1x 4 digit 7 segment display

4x BC327 pnp transistors

2x DALLAS DS18B20 temperature sensors

2x Red led

3x Button switch

   Resistors and wires
